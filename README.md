# Утилита создания документов 
Утилита для создания документов с помощью файла для [тестового задания](https://github.com/etherlowev/assignment)

Собранные утилиты лежат в директории output. Они собраны под разные архитектуры и операционные системы:
1. app_amd_windows
2. app_amd_linux
3. app_arm_linux
4. app_arm_darwin (macOS)

Пример использования: `app_amd_windows <Путь-к-файлу>`
